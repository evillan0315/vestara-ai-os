### Capabilities

- **Multi-model support** вҖ” Switch between local and cloud models mid-conversation
- **Voice input/output** вҖ” Speech-to-text and text-to-speech
- **Vision** вҖ” Analyze images, screenshots, documents
- **Memory-aware** вҖ” Remembers past conversations, user preferences
- **Context-aware** вҖ” Knows about your projects, files, and active tasks
- **Tool use** вҖ” Can execute terminal commands, search files, manage projects
- **Streaming** вҖ” Real-time response streaming
- **Markdown rendering** вҖ” Rich text, code blocks, tables, math
- **Code execution** вҖ” Run code snippets directly in conversation
